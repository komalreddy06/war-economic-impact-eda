# War Economic Impact Analysis

## Project Overview
This project analyzes the economic impact of wars using Python data analysis tools. 
The analysis explores relationships between variables such as military expenditure,
GDP change, and war duration.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook

## Analysis Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Correlation Analysis
5. Visualization (Heatmaps, Pairplots)

## Key Visualizations
- Correlation Heatmap
- Pairplot relationships
